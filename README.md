# 💗 Pink.dll (so-pink)

</div>

A **Seralyth plugin** that adds modules not yet made or not thought of.

> ⚠️ I, lucky, do not take accountability for bans you gain from using pink.dll.

---

## Additions

pink.dll adds the following mods into Seralyth

```
Client Mods
| Set name to 'Pinky'
| Set color to Pink
| Pink Name (cs)
| Pink Identifiers (cs)

Movement Mods
| Stable Bark Fly
| Front Flip

Master Mods
| VIM Kick Anti Report
| VIM Kick On Touch
| VIM Block Anti Report
| VIM Block On Touch

Misc Mods
| Splash On Touch [Removed soon > Added to Seralyth]
| Complete All Achievements
| Hoverboard Mess
| Rig Blind All
```

---

## Requirements

* [BepInEx](https://github.com/BepInEx/BepInEx)
* Gorilla Tag (PCVR)

---

## Installation

1. Install **BepInEx** for Gorilla Tag.
2. Download the latest `pink.dll` release.
3. Place `pink.dll` into:

```
Gorilla Tag/
 └── SeralythMenu/
     └── Plugins
```

4. Launch Gorilla Tag.
