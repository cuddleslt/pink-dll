# 💗 Pink.dll (so-pink)

A **Seralyth plugin** that adds modules not yet made or not thought of.

> ⚠️ I, lucky, do not take accountability for bans you gain from using pink.dll.

---

## Requirements

* [BepInEx](https://github.com/BepInEx/BepInEx)
* Gorilla Tag (PCVR)

---

## Installation

1. Install **BepInEx** for Gorilla Tag.
2. Download the latest `pink.dll` release.
3. Place `pink.dll` into:

```
Gorilla Tag/
 └── SeralythMenu/
     └── Plugins
```

4. Launch Gorilla Tag.
